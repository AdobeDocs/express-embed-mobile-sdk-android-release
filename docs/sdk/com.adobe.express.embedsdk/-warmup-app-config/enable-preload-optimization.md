# enablePreloadOptimization

[androidJvm]\
val [enablePreloadOptimization](enable-preload-optimization.md): [Boolean](https://kotlinlang.org/api/latest/jvm/stdlib/kotlin-stdlib/kotlin/-boolean/index.html) = false

Enable preload mechanism for container reuse optimization. When true: loads target URL with preload=true parameter. When false: loads traditional warmup/preload routes.
