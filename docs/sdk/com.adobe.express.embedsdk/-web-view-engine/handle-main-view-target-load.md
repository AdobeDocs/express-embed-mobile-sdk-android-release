# handleMainViewTargetLoad

[androidJvm]\
open override fun [handleMainViewTargetLoad](handle-main-view-target-load.md)()

Hint the compositor that the main WebView should be considered visible by clipping a small strip from the loader so the main is not fully occluded. Call this when the target reports DID_TARGET_LOAD.
